# array 形状配置(shape manipulating)

## 输入

```python
import numpy as np

a = np.floor(10*np.random.random((3,4)))
#floor,用于四舍五入

print(a,"this is  the array","-"*50,"\n")

print(a.shape, "this is the shape of the array.","-"*50,"\n")#返回shap

print(a.ravel(),"print the array in single dimension format.","-"*50,"\n")#以一维形式返回array

print(a.reshape(3,4),"reshape the array as the paramaters you type in.","-"*50,"\n")#reshape

print(a.T,"tranforming the array.","-"*50,"\n")#a dot following a capital T for transforming
```

## 输出

```python
[[9. 9. 9. 7.]
 [8. 6. 7. 3.]
 [1. 3. 4. 8.]] this is  the array -------------------------------------------------- 

(3, 4) this is the shape of the array. -------------------------------------------------- 

[9. 9. 9. 7. 8. 6. 7. 3. 1. 3. 4. 8.] print the array in single dimension format. -------------------------------------------------- 

[[9. 9. 9. 7.]
 [8. 6. 7. 3.]
 [1. 3. 4. 8.]] reshape the array as the paramaters you type in. -------------------------------------------------- 

[[9. 8. 1.]
 [9. 6. 3.]
 [9. 7. 4.]
 [7. 3. 8.]] tranforming the array. --------------------------------------------------
```

# array的拼凑

## 输入

```python
import numpy as np

a = np.random.random((2,2))
b = np.random.random((2,2))

v_stack =  np.vstack((a,b))
#竖直方向上拼接
h_stack =  np.hstack((a,b))
#水平方向上拼接
print(v_stack,"\n"*2)
```
